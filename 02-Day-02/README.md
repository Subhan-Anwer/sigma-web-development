# Useful VS Code Extensions

### [1. Auto Rename Tag ](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag "Auto Rename Tag Extension")

Auto rename paired HTML/XML tag. Reducing your hassle to rename the opening and closing tag separately.


### [2. Live Preview - Microsoft ](https://marketplace.visualstudio.com/items?itemName=ms-vscode.live-server "Live Preview Extension")

Live Preview lets you run a local server so you can view your web pages in real time without needing to reload after every change.


### [3. Prettier - Code formatter ](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode "Prettier Extension")

Prettier formats your code beautifully so its readable and consistent.


### [4. Dracula Theme ](https://marketplace.visualstudio.com/items?itemName=dracula-theme.theme-dracula "Dracula Theme Extension")

Dracula Theme is my personal favourite theme for VS Code. There are hundreds of other themes that you can try according to your taste.


### [5. VS Code Icons ](https://marketplace.visualstudio.com/items?itemName=vscode-icons-team.vscode-icons "VS Code Icons Extension")

VS Code Icons Extension adds icons to your files and folders making it more clear and attractive.

## Link CSS File to HTML File

To link a CSS file in HTML, we use `<link>` tag.
For CSS connecting, use the link tag inside the `<head>` tag.

```bash
<link rel="stylesheet" href="style.css">
```

---

## Link JavaScript File to HTML File

To link a JavaScript file in HTML, we use `<script>` tag inside and bottom of the `<body>` tag.

```bash
<script src="script.js"></script>
```
