## HTML Heading

HTML headings are written with `<h1></h1>` all the way to `<h6></h6>`

`<h1>` defines the most important and biggest heading. `<h6>` defines the least important and smallest heading:

```bash
<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>
```

### Output:

# This is heading 1
## This is heading 2
### This is heading 3
#### This is heading 4
##### This is heading 5
###### This is heading 6

## HTML Paragraphs

HTML paragraphs are defined with the `<p>` tag:

**Example:**
```bash
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
```

---

## HTML Links

HTML links are defined with the `<a>` tag, also known as the anchor tag:

**Example:**
```bash
<a href="https://www.google.com">This is a link</a>
```

The link's destination is specified in the href attribute.

> If you want to open the link in new tab, theres a attribute for that which is `target`. give it `_blank` 

**Example:**
```bash
<a href="https://www.google.com" target="_blank">This is a link</a>
```
