## Inline Elements (Tags)

Inline elements do not start on a new line and only take up as much width as necessary.

Some examples of Inline Elements are:
- `<span>`
- `<a>`
- `<img>`
- `<input>`
- `<button>`
- `<label>`
- `<input>`

## Block-level Elements (Tags)

Block level elements are those who start on new line and take up the entire width of their container by default.

They essentially claim all the horizontal space for themselves, pushing any content that comes after them to a new line.

Some examples of Block-level Elements are:
- `<div>`
- `<p>`
- `<h1>` - `<h6>`
- `<ul>` & `<ol>`
- `<hr>`
- `<form>`
- `<video>`

#### Q: Can we Change the Property and make Block-level Elements into Inline Element and vice versa?

**Yes!** we can make Block-level Elements into Inline Element and vice versa with the help of css by changing the display value of a tag.
