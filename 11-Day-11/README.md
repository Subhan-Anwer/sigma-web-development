## Semantic Tags in HTML

### What are Semantic Tags?

Semantic tags are for **organizing & arranging** your code for crawlers, web browsers and to understand your main content easily and to use features like read aloud effeciently as well as enhancing the readability of your code.

### Benifits of using Semantic Tags

Using HTML semantic tags can greatly benifit both your website's **SEO and maintainability** of your code. They offer a way to structure your HTML in a meaningful manner, making your website more accessible and easier to understand.

### Example of Semantic Tags

1. `<nav>`: used for creating the navigation bar of the website.
1. `<header>`: container for all header content e.g navbar, logo, etc.
1. `<footer>`: container for footer content.
1. `<section>`: divide the each section of the page with section tag.
1. `<main>`: holds the actual main content of the website, between the header and footer, parent of all sections.
