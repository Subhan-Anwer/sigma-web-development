Viewport Units

From Wikipedia, the free encyclopedia

A viewport is a polygon viewing region in computer graphics.

Viewport units are a powerful tool for creating responsive designs. They allow you to size elements based on the size of the viewport, which can help ensure that your design looks good on all devices.

For example, you can use vw to set the width of an element to be a percentage of the viewport width. This can be useful for creating fluid layouts that adapt to different screen sizes.

Similarly, you can use vh to set the height of an element to be a percentage of the viewport height. This can be useful for creating full-screen sections or ensuring that content fits within the visible area of the screen.
