### Create Your First HTML File

Create a New Folder & Open it in VS Code. 

Click on the **New File** icon, name it whatever you want but its important that you give it .html extension.

For example: **index.html**

#### Generate Bolier Plate Code


inside the file write a single excalamtion mark " ! " and hit enter.

You will see the following boiler plate code of HTML, Dont worry about it, its a starting code of HTML.
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```

## How Websites Work?

You are a **Client** and you use **Browser** to access a website.
You write `youtube.com` and hit enter. The Browser takes your request to server to get the `youtube.com` website. The server then return the code and files of `youtube.com` to browser and the browser renders the website.

![Image Illustration of how websites work](how-websites-work.png "Image Illustration of how websites work")

Websites have two Parts, The **Frontend** and **Backend.**

---

## How Frontend Works

Frontend runs on user's browser (client-side) using **HTML** for structure, **CSS** for styling, and **JavaScript** for interactivity, transforming designs into functional, responsive web pages
